"Flexnav" is a fully responsive jQuery menu/navigation plugin. It was built with Less CSS which makes it easy to modify.

Features

  • Slide pattern responsive navigation/menu, space-saving, sexy as hell.
  • Build with Less CSS. Easy to modify, just change the value of variables.
  • Light and Dark theme plus multiple multi-colors support (with 9 preset color schemes) Easy to use.
  • Build with jQuery, with several options avaiable to change. Such as responsive breakpoint, animation speed.
  • Desktop style and Mobile style will not affect each other. So you can change them seperately, say you change the horizontal menu style to vertical style.
  • Well documented, with examples.
  • Work well with Boostrap. Test on Bootstrap 2.3.2 and Bootstrap 3.
